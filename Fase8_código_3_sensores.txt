El código controla un robot con tres sensores ultrasónicos (frontal y laterales) y dos servos para navegación inteligente. Permite detectar obstáculos, escanear el entorno y ajustar la dirección según las distancias medidas. Tardamos aproximadamente una semana en aplicar el código de forma efectiva, especialmente para que los sensores auxiliares funcionaran correctamente, afinando pines, calibraciones y lógica de decisión. El sistema evita colisiones laterales, realiza barridos panorámicos y toma decisiones como girar o retroceder según el ángulo más despejad
