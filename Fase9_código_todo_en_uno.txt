Este código controla un carro con Arduino que puede detectar obstáculos con sensores ultrasónicos, esquivar paredes, y reaccionar a colores detectados por una ESP32-CAM. Usa un botón para alternar entre tres modos: esquivar paredes, actuar según colores, y detenerse. Controla un motor con puente H, servos para dirección y escaneo, y un LED RGB para indicar el estado con colores (rosa, verde, naranja, turquesa, etc.). El código implementa antirrebote para el botón, escaneo de obstáculos, y respuestas específicas a colores como rojo, verde, azul, y magenta mediante comunicación serie desde la ESP32.
