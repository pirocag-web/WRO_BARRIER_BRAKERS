Este código controla un robot con Arduino usando un botón. Al iniciar, el LED se enciende azul y espera acción. Cada vez que se presiona el botón, cambia de estado: rojo, naranja o verde. En rojo, ejecuta la función todo(), que hace que el robot avance, detecte obstáculos, escanee el entorno, decida hacia dónde girar y lea colores. Para crear la función todo, se eliminó el void loop y se creó void todo() para que el interruptor funcione de forma efectiva. También incluye protección contra rebotes del botón para evitar lecturas falsas.
