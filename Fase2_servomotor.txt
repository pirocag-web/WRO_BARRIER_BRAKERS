El código permite tomar decisiones autónomas de giro y reajustar la dirección para continuar el recorrido. Esta fue una sesión de prueba para validar el comportamiento del vehículo al integrar servomotores para dirección y escaneo. El carro avanza hasta detectar un obstáculo, se detiene, realiza un barrido panorámico entre 0° y 170°, y gira hacia el ángulo con mayor espacio libre. Si no hay salida clara, retrocede brevemente. Durante la prueba surgieron errores: algunas funciones estaban mal nombradas y al girar a la derecha el carro se regresaba, lo que evidenció fallos en la lógica de dirección, pero pudimos resolverlo de manera efectiva.
El sensor contaba con un problema, estaba demasiado alto como para detectar la pared y escanear su entorno y se tuvo que bajar la altura editando la estructura del carro para que no estuviera muy arriba el sensor.
